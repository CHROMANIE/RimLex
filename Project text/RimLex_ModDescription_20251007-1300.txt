RimLex – UI Text Localizer（Mod 説明 / 配布文面）
更新: 2025-10-07 13:00 (JST) / バージョン: 0.9.0（MVP）

■ 概要
RimWorld の画面に“表示された”英語UI文字列を収集し、辞書（strings_ja.tsv）に基づいて実行時に日本語へ置換するツール系MODです。
翻訳キーが無い海外製MODでも、自然な日本語UIで遊べる環境を作れます。

■ 特徴
- **翻訳済みテキストは再収集しないスマートフィルタ**（辞書ヒット時は収集せず即置換）
- **Mod設定メニューから操作**：辞書再読込／未訳抽出／整理（集約）／リセット／ログ／動作確認
- **Harmony による安全なUI層フック**で、軽量・高互換（署名差異に強い“ゆるい探索”）
- **Per-Mod 出力 + _All 集約**（未訳抽出・Mod別一覧・1列TSV）
- **ツールチップ対応**：TipRegion(string / TipSignal) の両系統を捕捉
- **UTF XML 生成（将来拡張）**：UTFBuilder をオプションとして外付け想定（現行はデフォルト無効）

■ 出力
- Export/PerMod/<Mod>/texts_en.txt / strings_en.tsv
- Export/_All/texts_en_aggregate.txt / untranslated.txt / strings_en_aggregate.tsv / grouped_by_mod.txt
- ログ: RimLex.log（INFO/WARN、件数サマリ）

■ 設定（RimLex.ini）
- ApplyDictAtRuntime / ExportPerMod / EmitAggregate / ExportMode / PerModSubdir / MinLength / ExcludePatterns
- DictPath / LogPath / ExportRoot（%ModDir% 変数対応）

■ サポート
- RimWorld 1.5 / 1.6, .NET Framework 4.7.2, Harmony 2.x
- 依存MODなし
